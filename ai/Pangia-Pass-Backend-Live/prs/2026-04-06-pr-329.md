## Key Changes

### Voice Infrastructure
- Telnyx webhook handler with ED25519 signature verification (PyNaCl)
- Voice configuration stored per-number in Firebase RTDB
- Call records with status tracking (ringing → answered/declined/missed/completed)

### API Endpoints
- `POST /api/v1/telephony/voice/call` - Initiate outbound call
- `POST /api/v1/telephony/voice/call/{callId}/answer` - Answer inbound call
- `POST /api/v1/telephony/voice/call/{callId}/decline` - Decline inbound call
- `POST /api/v1/telephony/admin/voice/config` - Admin voice config management

### FCM & Notifications
- VoIP push notifications for incoming calls (iOS CallKit, Android ConnectionService)
- Missed call notifications after 30s timeout via Cloud Tasks

### Performance
- O(1) call_control_id indexed lookup in Firebase (vs O(n) scan)
