# Pangia App - Repository Summary

## Overview

Pangia App is a React Native (Expo SDK 54) iOS mobile application that provides VoIP calling, SMS messaging, and phone number management. Built for True AI Inc, it targets international users who need provisioned phone numbers with voice and text capabilities.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Language | TypeScript 5.9.2 |
| Framework | React Native 0.81.5 via Expo SDK 54 |
| Navigation | React Navigation 7 (bottom tabs, native stack) |
| State | React Context (Auth, Call, Plan, Theme) |
| Storage | AsyncStorage (local), Firebase RTDB (cloud) |
| Auth | Firebase Authentication |
| VoIP | Telnyx WebRTC SDK + CallKit + PushKit |
| Push | Firebase Cloud Messaging (FCM) |
| Backend | Firebase Cloud Functions (Node.js 20) |

## Key Features

- **Voice Calling**: Outbound/inbound calls via Telnyx WebRTC with native iOS CallKit UI
- **SMS**: Send/receive messages with inbox, quota tracking, pagination
- **Phone Numbers**: Search, provision, and release numbers by country/area code
- **Subscriptions**: Monthly, annual, and lifetime plans via Firebase
- **Push Notifications**: FCM for standard, PushKit for VoIP (background wake)
- **Home Screen**: Sunrise/sunset visualization with SunCalc, plan cards
- **Theming**: Light/dark mode (system or manual toggle)
- **Dev Tools**: In-app dev menu, browser console, API tester CLI, remote logging

## Database

Firebase Realtime Database:
- `users/{uid}/profile` - User profile data
- `users/{uid}/fcmToken` - FCM push tokens
- `users_secure/{uid}/subscription_status` - Plan type, status, expiry

## API Endpoints (Backend REST)

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/health` | GET | Health check |
| `/api/v1/telephony/numbers/search` | POST | Search available numbers |
| `/api/v1/telephony/numbers/provision` | POST | Provision a number |
| `/api/v1/telephony/users/{id}/numbers` | GET | List user's numbers |
| `/api/v1/telephony/numbers/{id}/release` | POST | Release a number |
| `/api/v1/telephony/sms/send` | POST | Send SMS |
| `/api/v1/telephony/sms/quota` | GET | Get SMS quota |
| `/api/v1/telephony/sms/inbox` | GET | Get SMS inbox |
| `/api/v1/telephony/fcm/register` | POST | Register FCM token |
| `/api/v1/telephony/voice/call` | POST | Initiate voice call |
| `/api/v1/telephony/voice/telnyx/webrtc-token` | GET | Get WebRTC credentials |
| `/api/v1/telephony/voip/register-token` | POST/DELETE | Manage VoIP push token |
