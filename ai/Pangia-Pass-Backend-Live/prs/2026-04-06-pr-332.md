## Description

## Summary

Implements complete backend infrastructure for Telnyx built-in voicemail support, enabling users to receive, store, and manage voicemail recordings when calls are not answered.

## Implementation Details

### Core Features
- **Voicemail Service**: Download recordings from Telnyx (10-min URL expiry), upload to Firebase Storage for permanent retention
- **REST API**: Full CRUD operations (list, get, mark read, delete) with ownership validation
- **Webhook Handler**: Processes `calls.voicemail.completed` events from Telnyx
- **FCM Notifications**: Push notifications for new voicemails with caller ID and duration
- **Feature Flag**: `TELEPHONY.VOICEMAIL_ENABLED` for gradual rollout
