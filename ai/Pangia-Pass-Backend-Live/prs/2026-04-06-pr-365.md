## Summary

- Implements GCP Cloud KMS envelope encryption (Tink + AES-256-GCM) for SMS bodies at rest in Firebase RTDB
- Encrypts on all 4 write paths in `sms_service.py`, decrypts on read in inbox API with parallel batch decryption via ThreadPoolExecutor
- Adds `SMSEncryptionService`, `PrimitiveCache` (DEK caching), and `MessageEncryptor` singleton with graceful degradation fallback
- Includes Terraform KMS infrastructure (`kms.tf`), config integration via Secret Manager, and `tink[gcpkms]` dependency
