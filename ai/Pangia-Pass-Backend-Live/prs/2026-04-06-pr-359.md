## Summary
- Refactors telephony webhooks from CloudNumbering-specific endpoints to generic provider-agnostic dispatcher (`/webhooks/{provider}/sms/inbound`, `/webhooks/{provider}/sms/delivery`)
- Extracts webhook verification into dedicated `webhook_security.py` module with per-provider verification strategies (IP allowlist for CloudNumbering, HMAC for Telnyx)
- Removes CloudNumbering-specific fields from DB schemas, making them provider-neutral
- Enhances `TelephonyFactory` with capability-based provider registration and dynamic initialization
- Extends base provider class with `verify_webhook()` contract
- Adds CloudNumbering inventory management module
